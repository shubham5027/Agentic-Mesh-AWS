# Contributing to Agentic Mesh

Thank you for your interest in Agentic Mesh! We welcome contributions to help improve the system.

## How to Contribute

1.  **Fork the Repo**: Create your own copy of the project.
2.  **Create a Branch**: Use a descriptive name like `feat/new-agent`.
3.  **Implement Changes**: Ensure your code follows the established style.
4.  **Tests**: Run existing tests and add new ones if applicable.
5.  **Pull Request**: Submit your changes for review.

## Code Style
- Follow PEP 8 for Python.
- Use meaningful commit messages.
- Keep components modular and serverless-friendly.

We appreciate every star and pull request! ⭐
