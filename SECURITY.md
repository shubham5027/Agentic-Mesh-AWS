# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities by creating a Private Vulnerability Report on GitHub or by contacting the maintainer.

Do not open public issues for security vulnerabilities until they have been addressed. We aim to respond within 48 hours and provide a fix within 7 days.
